Brewlogger

A web application designed for coffee lovers everywhere, from the professional barista to the home coffee enthusiast. This cutting-edge React project can be accessed from any device with a web browser, providing users with the flexibility to log their coffee brews and replicate their favorite coffee beverages from anywhere.

With a focus on user-friendliness and performance, the application was built using the latest technologies in web development, including React and Tailwind CSS. This allows for an engaging and interactive user interface that is both aesthetically pleasing and functional. To ensure fast and reliable data management, the application leverages Firebase, providing real-time updates and secure authentication for users.

Tools: React, Tailwind CSS, Firebase

BrewRecipes REST API

An API with accessible coffee recipes for use on any frontend. This Express backend project has full CRUD functionality and connects to MongoDB for storing data. Users can make GET, PATCH, POST, and DELETE requests using the Fetch API or their favorite tool like Axios.

Tools: Express, MongoDB, Mongoose ODM
